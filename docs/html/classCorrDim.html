Detailed Description

Class responsible for evaluation of the correlation dimension of the input data. This class is purposefully made generic to handle the data of any dimension. Go through the API documentation for more details.

Usage: CorrDim d = CorrDim(my_data, num_data, data_dim); printf("Correlation Dimension = %d\n", d.evalCorrDim(10)); printf("Correlation Dimension = %d\n", d.evalCorrDim(20));

It is the responsibility of the calling function to allocate and free the memory occupied by 'log_cr', 'log_r' and 'inter'! 'min_logr' and 'max_logr' should be in the range (-INF, 0]. The reason being the distances would have been normalized to the range [0, 1], implying that for comparison, the range you specify must be in (-INF, 0] This has been done mainly for the sake of simplicity and uniformity for different types of maps.
